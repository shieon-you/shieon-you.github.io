• Readium LCP and ISO TS 23078-2 are technically same standards. ISO/IEC JTC1/SC34 committee resolved in September, 2018 to start an ISO standardisation work, where an industrial standard, Readium LCP goes to an ISO Technical Specification. The ISO document number is ISO TS 23078-2.
  • Final release schedule of the Technical Specification is scheduled in 2020.
