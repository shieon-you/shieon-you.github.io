  • Readium LCP(Licensed Content Protection) is an industrial standard for eBook DRM released by Readium Foundation in 2016. The specification of Readium LCP is to an ISO standard(ISO TS 23078-2) now in progress at ISO/IEC JTC1/SC34 committee from 2018.
  • It is passphrase based DRM. So anyone who knows the passphrase can copy the protected eBook into his/her or friendsâ€™ other device.
  • In Readium LCP, user's phone number, site id or e-mail is can be stored in the protected eBook so that if the protected eBook is uploaded into internet with passphrase, the first distributor can be detected easily.
  • OPENS DRM provides two types of DRM, Readium LCP(ISO TS 23078-2) and device based protection (ISO TS 23078-3).
  • Readium LCP and ISO TS 23078-2 are technically same standards. ISO/IEC JTC1/SC34 committee resolved in September, 2018 to start an ISO standardisation work, where an industrial standard, Readium LCP goes to an ISO Technical Specification. The ISO document number is ISO TS 23078-2.
  • Final release schedule of the Technical Specification is scheduled in 2020.
  • ISO TS 23078-2 is an ISO standard which is technically based on the Readium LCP. It is user key based protection technology, with which eBooks are protected and delivered to specific user who knows the associated passphrase.
  • On the other hand, ISO TS 23078-3 is an ISO standard which is technically based on device key based protection. With this type, eBooks are protected and delivered only to specific number of devices which are allowed by service provider.
